tools

generative motion tools. each one runs entirely in the browser, is seeded and deterministic, and exports a still or a video. source is public.

~ flow fields.

convergence flow fine colored lines pinch through one central point and fan out to visible endpoints. src flux knot braided strands accelerate through a knot — compressing on approach, stretching into streaks on exit. src path packets colored packets routed along procedural curves, branching toward destination ports. src signal constellation fibers converge into pipelines, branch into channels, emerge as particles receding into depth. src prism flow procedural spectral motion on a custom webgl renderer. src

~ type & raster.

glyph flow a micro-type texture revealed by a moving organic field: flowing masses, eroded voids, sparse dust. src pixel sweep a phrase repeating in rows, each phase-offset, riding a side-to-side sine wave. src halftone sweep a circular grid of squares toggling on and off as a selectable sweep loops across it. src

~ how they're built.

react, vite and typescript, rendered on canvas or webgl. every output is a pure function of a seed and a timeline position, so the same seed always gives the same frame. clone any repo and run npm install, then npm run dev.
